解决快捷键冲突

问题描述:

今天在使用IDEA的设置代码格式快捷键不好用，也检查了快捷键设置没有问题，只能是快捷键冲突，但因为软件比较多不想一个个软件筛查，就开始看各种论坛，问ai，终于找到一款懒人专用的解决快捷键占用冲突问题。

OpenArk官网

OpenArk官方使用文档

OpenArk64 1.5.0下载

使用方法

点击内核–>系统热键–>进入内核模式

输入你想查找占用的快捷键

选中右键，可以定位，也可以删除

小注：

输入法的快捷键是单独管理的，在这里是无法找到的。
